# Pricing Guide

These are starting points. Final quotes require sample review.

## E-commerce Product Image Package

- Trial: 3 SKUs for sample approval
- Batch: 100 SKUs or more
- Deliverables: JPG/PNG images, delivery manifest, failed-item report
- Internal reference: 100 SKUs starts around CNY 4,300

## Offline Text/Data Processing

- Trial: 100 rows for schema and accuracy validation
- Batch: 100,000 rows or more
- Deliverables: CSV/Excel/JSON, failed-row report, sample audit
- Internal reference: 100,000 rows starts around CNY 6,500

## RAG Knowledge Base Initialization

- Trial: 20-100 documents
- Batch: priced by document count, page count, parsing complexity, and evaluation set
- Deliverables: cleaned chunks, vector store, QA report, reproducible setup
- Internal reference: 300 documents starts around CNY 10,400

## Cost Basis

Current compute cost assumptions:

- NVIDIA L40: CNY 2.5 per GPU hour
- NVIDIA A800: CNY 5 per GPU hour
- AMD CPU node: CNY 0.04 per core hour
- Intel CPU node: CNY 0.06 per core hour
- High-memory node: CNY 0.08 per core hour

Compute cost is usually not the main expense. Scoping, data cleaning, QA, revisions, and customer communication dominate total project cost.
