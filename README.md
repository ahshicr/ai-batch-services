# GPU Batch AI Services

This repository is a public service desk for GPU-powered batch AI tasks.

## Services

- E-commerce product image generation
- Batch data extraction, classification, and cleanup
- RAG knowledge base initialization from company documents
- LoRA/QLoRA fine-tuning preparation, training, and evaluation

## Workflow

1. Open an issue using one of the service request templates.
2. Share a small sample dataset or a few product images.
3. We confirm scope, output format, acceptance criteria, and quote.
4. A sample batch is delivered first.
5. After approval and payment confirmation, the full batch is processed.
6. Final outputs are delivered as files, reports, model artifacts, or a reproducible demo.

## What We Do Not Offer

- Raw GPU rental
- 24/7 hosted API service
- Long-term cloud desktop or remote workstation rental
- Work involving illegal, deceptive, abusive, or unauthorized data usage
